## 🎯 What Is Prism?

**Prism refracts complexity into clarity.**

A one-command tool that sets up production-ready development environments with:

✨ **Beautiful web UI installer** (or CLI if you prefer)  
📦 **npm-distributed config packages** (fetch from unpkg CDN!)  
📚 **Auto-generated documentation site** (MkDocs with branding)  
💼 **Career management tools** (goals, wins, 1-on-1s, reviews)  
🛠️ **Complete tool installation** (Git, Docker, cloud CLIs, IDEs)  
📁 **Smart folder structure** (projects, experiments, learning, archive)  
🔄 **Resume capability** (pick up where you left off)  
🎭 **7 pre-built packages** (personal, startup, enterprise, university, etc.)

---

## 🚀 Key Innovation: npm Package Distribution

**All config packages are distributed via npm public registry!**

This is a **GAME CHANGER** because:

✅ **No custom infrastructure** - Uses npm's global CDN  
✅ **No npm install required** - Fetches directly from unpkg.com  
✅ **Fast global delivery** - CDN distributed worldwide  
✅ **Works offline** - Falls back to local packages  
✅ **Discoverable** - Search on npmjs.com  
✅ **Versioned** - Semantic versioning built-in  
✅ **Open ecosystem** - Anyone can publish @prism packages

### Published Packages:

- `@prism/personal-dev-config` - Freelancers & indie developers
- `@prism/startup-config` - Startups (fast, lean, agile)
- `@prism/fortune500-config` - Large enterprises
- `@prism/university-config` - Research & teaching
- `@prism/consulting-config` - Multi-client consulting
- `@prism/opensource-config` - Open source projects
- `@prism/acme-corp-config` - Template for companies

**Each package includes:**
- `package.json` (npm metadata)
- `package.yaml` (Prism configuration)
- `README.md` (documentation)
- `config/` (tool configs)
- `assets/` (logo, colors, branding)

### Cross-Platform

✅ macOS (Intel & Apple Silicon)  
✅ Windows 10/11  
✅ Linux (Ubuntu, Debian, Fedora)

### Package Fetching Strategy

```
1. Try unpkg CDN (https://unpkg.com/@prism/PACKAGE@latest/package.yaml)
   ↓ if fails
2. Try npm registry directly
   ↓ if fails
3. Use local config-packages/ directory
   ↓ if fails
4. Show error with manual download instructions
```

## 💼 Usage Examples

### Quick Start

```bash
# Clone
git clone https://github.com/andersonwilliam85/prism.git
cd prism

# Web UI (recommended)
python3 web_installer.py
# Opens http://localhost:5000 automatically

# OR CLI
python3 install.py
# Follow interactive prompts
```

### Package Management

```bash
# List packages (from npm)
python3 scripts/npm_package_fetcher.py list

# Fetch specific package
python3 scripts/npm_package_fetcher.py fetch @prism/startup-config

# Use local packages only
python3 scripts/npm_package_fetcher.py list --local
```

### Publishing Packages

```bash
# Test first
python3 scripts/publish_packages.py --package personal-dev --dry-run

# Publish
python3 scripts/publish_packages.py --package personal-dev

# Publish all
python3 scripts/publish_packages.py --all
```
