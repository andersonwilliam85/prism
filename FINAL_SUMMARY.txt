WHAT MAKES THIS SPECIAL?
================================================================================

1. npm Distribution - First dev env tool to use npm as package registry!
   • No custom infrastructure needed
   • Global CDN via unpkg.com
   • Works without npm installed
   • Anyone can publish @prism packages

2. Beautiful Web UI - Most setup tools are CLI-only
   • Visual package selection
   • Step-by-step wizard
   • Live progress updates

3. Career Management - Built-in professional development tools
   • Goal tracking
   • Win logging
   • 1-on-1 prep
   • Performance reviews

4. Offline Capable - Works without internet
   • Fetches from unpkg CDN when online
   • Falls back to local packages offline

5. Extensible - Easy to create custom packages
   • 7 pre-built packages included
   • Template package for companies
   • Hierarchical config inheritance
